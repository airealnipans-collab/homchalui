## hreflang
- Emit alternates **only for locales with a published translation**, plus `x-default` → Thai.
- Self-referencing canonical per localized page (no cross-locale canonical).

## Sitemaps
- `/sitemap.xml` = index → `/sitemap-th.xml`, `/sitemap-en.xml`, `/sitemap-zh.xml`.
- Each lists only published localized URLs; include `xhtml:link` hreflang.
- Regenerated by the worker on schedule and on publish.

## Localized metadata & schema
Title, description, OG, Twitter, FAQ, and JSON-LD are localized; set `inLanguage` correctly.
Never emit metadata/sitemap/hreflang for unpublished/untranslated localized pages, and never
show Thai content on `/en`/`/zh`.

## URL rules
Thai no prefix; `/en`, `/zh`; per-locale slugs; never `/th`.

## Common pitfalls (avoid)
- Thai fallback leaking into `/en`/`/zh`. - Duplicate canonical across locales. - hreflang to
  unpublished pages. - Reusing Thai slug under other locales.
