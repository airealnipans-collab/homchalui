# Homchalui Production Logo Kit

This package contains production-ready raster assets generated for the หอมฉลุย brand.

## Included
- `logo-horizontal.png`
- `logo-stacked.png`
- `logo-icon.png`
- `logo-icon-dark.png`
- `app-icon-1024.png`
- `app-icon-512.png`
- `android-chrome-192.png`
- `apple-touch-icon.png`
- `og-logo-banner.png`
- `favicon.ico` and favicon PNG sizes
- `brand-identity-board.png`

## Recommended usage
- Desktop header: `logo-horizontal.png`
- Mobile header: `logo-icon.png` or `logo-stacked.png`
- Dark backgrounds: `logo-icon-dark.png`
- Social sharing/Open Graph: `og-logo-banner.png`
- App / PWA icon: `app-icon-512.png` or `app-icon-1024.png`
- Browser favicon: `favicon.ico`
