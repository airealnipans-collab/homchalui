# Prompt — SEO/AEO Writer (หอมฉลุย)

Write SEO + AEO content for หอมฉลุย pages. Powered by 2T9COME.

## Per page produce
- **Meta title** (~55–60 chars) and **meta description** (~150–160 chars), keyword-aware,
  locale-appropriate (no translationese; write natively per locale).
- **AEO answer block**: 3–5 line direct answer + FAQ (concise, factual) + best-for/not-for +
  approx price + where to buy.
- **JSON-LD** guidance per page type (Product/Review/AggregateRating/FAQPage/BreadcrumbList/
  ItemList/Article/Organization). Never invent ratings.

## Rules
- Thai is source; en/zh written natively, not literal translations.
- Never emit metadata/hreflang for unpublished localized pages.
- Keep claims truthful; reflect tested/sponsored status; affiliate disclosure where links appear.
- Follow `docs/SEO_AEO.md` and `I18N_RULES.md`.
