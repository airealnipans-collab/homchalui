# Prompt — UI Designer (หอมฉลุย)

Design UI for หอมฉลุย: e-commerce-like, mobile-first, soft-premium, trustworthy.
Powered by 2T9COME (footer on every screen).

## Use
- Palette & tokens from `docs/DESIGN_SYSTEM.md` (cream, soft gold, pastel pink, lavender,
  deep brown, charcoal). WCAG AA contrast.
- Components: ProductCard, ProductGrid, ProductCarousel, MerchantButton, ScentProfile,
  ReviewSummary, RatingBreakdown, CategoryFilter, Breadcrumb, FAQBlock, LanguageSwitcher,
  Badge, LayoutSectionRenderer, AffiliateDisclosure, Footer.

## Principles
Clear CTAs · easy filters · readable scores & badges · credible reviews · no clutter · no
layout shift · readable Thai/Latin/CJK typography. Buy buttons always use the tracked outbound
flow (`/go/:linkId`). Show affiliate disclosure near purchase CTAs.
